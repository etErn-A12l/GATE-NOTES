

## In Computer Network - 

```
Data Size:
1 KB = 1024 bytes = 1024 * 8 bits = 8192 bits
1 MB = 1024 KB = 1024 * 1024 bytes = 1024 * 1024 * 8 bits = 8,388,608 bits

Bandwidth Size:
1 KB = 1000 bytes = 1000 * 8 bits = 8000 bits
1 MB = 1000 KB = 1000 * 1000 bytes = 1000 * 1000 * 8 bits = 8,000,000 bits

Diagram:

Data Size:
1 KB (8192 bits)   <---------------------->   1 MB (8,388,608 bits)

Bandwidth Size:
1 KB (8000 bits)   <---------------------->   1 MB (8,000,000 bits)

```
